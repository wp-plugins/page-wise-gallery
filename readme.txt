== Description ==

This plugin gives you provision for adding separate gallery for each page :
Use the shortcode [pwg_gallery] on your post or page
OR
<?php echo do_shortcode('[pwg_gallery]'); >? On your page template.
OR
You can add the pwg widget to the sidebar : Where you can set the size of the thumnail

== Installation ==

1. Upload `page-wise-gallery.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php echo do_shortcode('[pwg_gallery]'); ?>` in your templates
OR
<?php echo do_shortcode('[pwg_gallery]'); >? On your page template.
OR
You can add the pwg widget to the sidebar

== Frequently Asked Questions ==

= The pop up feature is not working for my page wise gallery =

Please make sure you have installed the Fancy box plugin (https://wordpress.org/plugins/easy-fancybox/)
